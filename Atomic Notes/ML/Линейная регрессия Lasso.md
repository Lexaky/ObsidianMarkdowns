# Линейная регрессия Lasso


#### Рассматриваемое знание
В линейную регрессию добавляется регуляризация.
Линейная регрессия + MSE + $l_1$ регуляризация (абсолютная)
$$MSE + \alpha ||\omega||_1$$
$||\omega||_1$ - это сумма элементов:
$$||\omega||_1 = \sum_{i=1}^n |\omega_i|$$ Эта линейная регрессия называется **Lasso**.
#### Эффекты
При увеличении параметра $\alpha$ модель будет занулять наиболее не интересные признаки. Мы можем увеличивать $\alpha$ и постепенно выключать те признаки, которые вносят наименьший вклад в результат.

#### L2 регуляризация
Линейная регрессия + MSE + $l_2$ регуляризация 
$$MSE + \alpha ||\omega||_2$$
$||\omega||_2$ - это $||\omega||_1$ под квадратным корнем:
$$||\omega||_2 = \sqrt {\sum_{i = 1}^n |\omega_i|^2}$$
Это называется **Ridge**. Особенности: не зануляет, а делает маленькими значения не интересных признаков.

#### Вывод
Lasso используется для исследования важности признаков. Например, при 100 признаках я хочу узнать, какие из них бесполезны - выбрасываю их и использую другие методы обучения.

Использование и $l_1$ и $l_2$ называется [[Линейная регрессия ElasticNet]]
